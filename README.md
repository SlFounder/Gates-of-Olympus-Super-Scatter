# Gates-of-Olympus-Super-Scatter
Gates of Olympus Super Scatter — улучшенная вариация классического слота от Pragmatic Play, где шанс выпадения Scatter заметно выше. Это приводит к более частым фриспинам, усиленным множителям и динамичному геймплею. Механики каскадов и множителей до x500 делают Super Scatter одной из самых взрывных версий знаменитого слота.
